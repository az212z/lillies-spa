# Lillies Spa — موقع تعريفي (سبا وصالون تجميل نسائي)

موقع تسويقي عربي (RTL) فاخر لـ **Lillies Spa** — سبا وصالون تجميل نسائي في المدينة المنورة (بئر عثمان).

## المعلومات
- **النشاط:** سبا وصالون تجميل نسائي
- **المدينة:** المدينة المنورة — بئر عثمان
- **التقييم:** 4.5 على خرائط قوقل (1,480 تقييم) — مذكور في الموقع وفي بيانات JSON-LD
- **الجوال / واتساب:** 0502226166 · 966502226166
- **الخدمات (بدون أسعار):** شعر، مكياج، أظافر، حمام مغربي وسبا، عناية بالبشرة، باقات المناسبات

## لماذا اختير هذا التصميم — "Lily Elegance"
هوية مستقلة عن المواقع الشقيقة (sage-zen / rose-gold). الفكرة قائمة على **زهرة الزنبق (Lily)** التي يحملها اسم المكان:
- **الألوان:** بنفسجي ليلكي ناعم + عاجي + ذهبي دافئ + برقوقي عميق للنص. أنثوي راقٍ دون مبالغة.
- **الخطوط:** عناوين **El Messiri** (أنيق، أنثوي)، والنص **Tajawal** (نظيف، واضح).
- **الإحساس:** floral-luxury هادئ، مساحات واسعة، حركة ناعمة.

## التقنية
- **Vanilla** بالكامل: HTML + CSS + JavaScript، **بدون أي build step** أو أطر عمل.
- خطوط Google عبر preconnect + `display=swap`.
- أيقونات SVG inline (بدون إيموجي).
- scroll-reveal عبر IntersectionObserver مع fallback أمان.
- قائمة جوال = **overlay ملء الشاشة معتم** بزر X واضح.

## حالة الـ backend
**غير مربوط بأي خادم.** نموذج الحجز تجريبي للعرض:
- يتحقق من الحقول، يحفظ البيانات محليًا في `localStorage` (مفتاح `lillies_bookings`)،
- ثم يبني رسالة واتساب جاهزة ويفتح `wa.me/966502226166`،
- مع إشعار (toast) للتأكيد.

## مصدر الصور
صور حقيقية من ملف العمل على خرائط قوقل، تمّت معاينتها وتنقيتها (curate): استُبعدت صورة عرض ترويجي تحوي سعرًا (لا أسعار مخترعة)، واعتُمدت 5 صور للأجواء والخدمات.

## التشغيل محليًا
```bash
cd lillies-spa
python3 -m http.server 4173
# ثم افتح http://localhost:4173
```
